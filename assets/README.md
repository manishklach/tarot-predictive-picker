# Assets Directory

Place the publication assets for the GitHub Pages site in this directory.

Expected files:

- `patent.pdf`
- `architecture-overview.png`
- `gather-flow.png`
- `favicon.png`

Optional directory:

- `figures/`

Recommended contents:

- `patent.pdf`: published patent or patent application PDF
- `architecture-overview.png`: clean publication-safe architecture figure
- `gather-flow.png`: execution or gather flow figure
- `favicon.png`: small square favicon for the site
- `figures/`: supplemental figures, exports, or diagram set

The site uses relative paths, so these files should be placed exactly in:

```text
/assets/patent.pdf
/assets/architecture-overview.png
/assets/gather-flow.png
/assets/favicon.png
/assets/figures/
```
